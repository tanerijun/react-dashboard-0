## React Dashboard

An admin dashboard

### Tech stack:

- TypeScript
- React
- MUI

### Features

- [x] Registration form
- [x] Light and Dark theme
- [x] Various charts
- [x] FAQ Accordion
- [x] Table with Data Grid
- [x] Calendar

### Local installation

1. Install dependencies

   ```
   pnpm i
   ```

2. Run dev server
   ```
   pnpm dev
   ```
